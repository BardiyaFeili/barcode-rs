# Barcode, a TUI Code Editor

Barcode is a simeple, modal, terminal code editor written in Rust.

## Features

- Fast
- Multiple tabs
- Easy to extend
- Cross-platfrom
- Modal, like Vim

## How to build

First clone the repo, then navigate to the directory in the terminal.

```bash
# To build
cargo build --release

# To run
Cargo run --release
```

## How to use

You can enter Insert mode by pressing `i`.
